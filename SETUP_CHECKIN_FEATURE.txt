INSTALLATION (3 SIMPLE STEPS)
===============================================

STEP 1: Import Database Table
------------------------------
Option A - Using phpMyAdmin:
1. Open: http://localhost/phpmyadmin
2. Select database: hostel_db
3. Click "Import" tab
4. Choose file: checkin_checkout_table.sql
5. Click "Go"

Option B - Using MySQL Command:
1. Open XAMPP Shell
2. Run: mysql -u root hostel_db < checkin_checkout_table.sql

STEP 2: Verify Installation
----------------------------
1. Login to admin panel
2. Look for "Check-in/Check-out" in sidebar menu
3. Click it to open the page

STEP 3: Test the Feature
-------------------------
1. Select a student from dropdown
2. Choose "Check-in (Entry)"
3. Click "Record Entry"
4. You should see success message!
